absolute position, center both vertically and horizontally

way1-1:absolute position: negative margin

this way uses absolute position and makes the content element out of the normal flow, which is not good for some situation. Other than that, it's a good way to use

  • doesn't require addtional objects
  • easy and simple
  • fit for all browsers
  • only for elements with fixed height and width
  • need 'overflow:auto' to avoid overflow
    .content{
        position: absolute;
        top: 50%;
        left: 50%;
        width: 240px;
        height: 240px;
        margin-top: -120px;
        margin-left: -120px;
        overflow: auto;
    }

way1-2:absolute position: negative transform

this way is similar to former way

    .content{
        position: absolute;
        top: 50%;
        left: 50%;
        transform: translate(-50%, -50%);
    }
  • doesn't need fixed height and width
  • doesn't require addtional objects
  • needs CSS3 support

way2: absolute position: stretching

'margin:auto' makes the center, extends the margin of content element to the whole parent

    .content{
        position: absolute;
        top: 0;
        left: 0;
        bottom: 0;
        right: 0;
        margin: auto;
        width: 240px;
        height: 240px;
        overflow: auto;
    }
  • fits for all browsers
  • only for elements with fixed height and width
  • needs 'overflow:auto' to avoid overflow
  • not for IE

centering vertically

way3:hidden floater

use a hidden floater with 50% height and negative half the height to vertically center the element. This is similar to way1-1 but without absolute position. Also the 'margin: 0 auto' makes the element horizontally centered

    #way3 .floater {
        float: left;
        height: 50%;
        width: 100%;
        margin-bottom: -120px;
    }

    #way3 .content{
        clear: both;
        height: 240px;
        width: 240px;
        /*the margin is to do the horizontal center*/
        margin: 0 auto;
        position: relative;
    }
  • fits for all browsers
  • need fixed height
  • need an addtional floater element

way4: display as a table

use table&table-cell display and vertical-align and we can vertically center the element

    #way4 {
        display: table;
    }
    #way4 .cell {
        display: table-cell;
        vertical-align: middle;
    }
  • does not need fixedd height
  • height is determined by the content
  • not for IE
  • needs additional elements

inline element

way5: inline ghost element

This is the describtion od the right demo. look at the picture below, we use a inline ghost element with 100% height, the vertical-align property makes all the inline elements vertically centered

  • does not need fixed height
  • maybe usable only with single children 
#way5 {
    text-align: center;
    white-space: nowrap;
}

#way5:before {
    content: '';
    display: inline-block;
    height: 100%;
    vertical-align: center;
}

#way5 .content {
    display: inline-block;
    vertical-align: middle;
    width: 300px;
}

way6: one-line element center

for inline elements, 'text-align: center' is enough to horizontally center the element. And the way vertically centering the element can be similar to the 'ghost element way' on the left. Especially, when there is only one line, we can equal the height and line-height to achieve the vertical center. 

    #way6 .content {
        height: 100px;
        text-align: center;
        line-height: 100px;
    }
